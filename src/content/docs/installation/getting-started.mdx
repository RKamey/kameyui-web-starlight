KameyUI es una librería de componentes para la creación de interfaces de usuario.

## Instalación

Para instalar KameyUI en tu proyecto, ejecuta el siguiente comando:

<Tabs>
  <TabItem label="npm">
    ```bash 
      npm install kamey-components
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash 
      pnpm install kamey-components
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash 
      bun add kamey-components
    ```
  </TabItem>
</Tabs>
