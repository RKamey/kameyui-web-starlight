KameyUI es una librería de componentes para la creación de interfaces de usuario.

## Instalación

Para instalar KameyUI en tu proyecto, ejecuta el siguiente comando:

<Tabs>
  <TabItem label="npm">
    ```bash 
      npm install kameyui
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash 
      pnpm install kameyui
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash 
      bun add kameyui
    ```
  </TabItem>
</Tabs>
